# WeRead IDE

微信读书笔记导出 Chrome 插件，导出格式支持 Markdown，方便导入各类笔记工具。

## 项目结构

```
weread-export-chrome/   # Chrome 扩展主项目
├── README.md
├── screenshot.png
├── manifest.json
├── background.js
├── popup.html / popup.js
├── sidepanel.html
├── content.js
└── icons/
```
